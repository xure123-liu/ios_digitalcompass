# 1. 文档说明

## 1.1 产品名称
- 中文名：数字指南针
- 英文名：Digital Compass（暂定）

## 1.2 产品定位
数字指南针是一款面向日常出行、户外导航、方向判断与宗教方位辅助场景的 iOS 工具类 App。产品通过数字角度读数、指南针圆盘、方向文本、精度提示、校准引导、麦加方向辅助与主题化界面，为用户提供打开即用、信息清晰、可持续使用的方向判断体验。

# 2. 产品定位与目标

## 2.1 产品形态
- 移动端工具类 App
- 首发平台：iOS
- 以单机使用为主，不依赖账号系统

## 2.2 目标用户
- 日常方向判断用户：需要快速查看东南西北和当前角度
- 户外出行用户：在徒步、露营、旅行等场景下辅助辨别方向
- 宗教方位用户：需要快速获取麦加方向并进行朝向辅助
- 工具型用户：偏好简洁直观、可长期使用的高频小工具用户

## 2.3 核心价值
- 提供实时方向文本与角度读数，降低方向判断成本
- 用高可读的圆盘与数字界面帮助用户快速识别当前朝向
- 提供精度状态、磁场干扰提示与校准流程，提升结果可信度
- 提供麦加方向能力，覆盖更明确的细分使用场景
- 通过订阅解锁主题、去广告和高级能力，形成可持续商业化路径

## 2.4 产品目标
- 成为一款打开即用、信息清晰、校准简单的数字指南针工具 App
- 在免费版满足基础方向查看需求的前提下，通过订阅提供更完整的高级体验
- 具备上架 App Store 所需的产品、商业化、法务和文案材料

# 3. 背景与机会

## 3.1 市场背景
- 指南针类产品属于高频、低学习成本的小工具，用户预期是打开即用、结果直观、无需复杂上手。
- 现有同类产品常见问题包括：界面陈旧、广告干扰明显、缺少精度表达、校准说明不清晰，以及在细分场景下缺少明确能力延展。

## 3.2 用户痛点
- 用户不仅需要“方向结果”，还需要知道当前读数是否可信、是否受磁场干扰、是否需要校准。
- 在户外或复杂环境中，方向结果抖动会直接影响判断效率。
- 宗教方位用户不仅需要基础指南针，还需要更直接的麦加方向指引，而不是自行换算。
- 工具类用户普遍反感高频广告与冗余流程，更偏好简洁、稳定、可长期使用的产品体验。

## 3.3 竞品机会点
- 多数竞品以基础圆盘 + 角度显示为主，缺少精度状态、校准链路和细分目标方向能力。
- 通过补足“精度提示 + 校准引导 + 麦加方向 + 主题化 + 订阅体系”，可以形成相对完整的工具产品闭环。

## 3.4 产品机会判断
- 通过高可读方向展示和可信度提示，产品可以在基础工具场景中建立稳定使用心智。
- 通过麦加方向等细分功能，可覆盖更明确的使用场景并提升差异化。
- 通过高级主题、去广告与高级样式形成 Pro 版权益，有机会建立轻量但可持续的工具类订阅模型。

# 4. 用户画像与场景

## 4.1 核心用户画像

### Persona A：日常出行用户
- 身份：城市出行、旅游、路面判断方向的普通用户
- 目标：快速确认当前朝向与角度
- 关注点：界面是否够直观、读数是否稳定、打开是否够快

### Persona B：户外活动用户
- 身份：徒步、露营、登山、钓鱼等户外用户
- 目标：在复杂环境中辅助判断方向
- 关注点：精度提示是否明确、校准流程是否顺畅、深色界面是否易读

### Persona C：宗教方位用户
- 身份：有麦加朝向需求的穆斯林用户
- 目标：快速查看相对麦加方向并完成朝向判断
- 关注点：方向是否清晰、入口是否直接、提示是否可理解

## 4.2 核心使用场景
- 场景 1：用户打开首页，查看当前方向文本和角度值
- 场景 2：设备受磁场干扰时，根据提示进行指南针校准
- 场景 3：用户切换到麦加方向页，查看当前位置相对麦加的方向指引
- 场景 4：用户在设置中切换主题、精度显示、声音反馈与语言
- 场景 5：用户点击高级主题或去广告权益时进入订阅页完成升级

## 4.3 关键行为路径
1. 用户打开 App
2. 系统检查方向传感器与定位可用状态
3. 首页显示当前方向、角度、精度状态与指南针圆盘
4. 如精度较差，用户点击“去校准”进入校准引导
5. 如有宗教场景需求，用户点击“麦加方向”进入对应页面
6. 如需主题、去广告或高级功能，进入订阅页购买月订或年订
7. 用户在设置页管理语言、主题、提示文案和协议入口

# 6. 核心模块与交互逻辑

## 5.1 模块 A：实时指南针
- **模块目标**：实时展示当前方向、角度与指南针圆盘
- **用户价值**：打开即用，快速识别当前朝向
- **优先级**：P0

### 功能点列表
- 当前方向文本显示
- 当前角度数值显示
- 指南针圆盘联动
- 精度状态展示

### 详细交互逻辑
1. 用户操作：打开 App，停留在首页指南针页。
2. 系统响应：读取设备方向传感器数据并实时计算朝向角度。
3. 数据流转：持续读取磁力计 / 陀螺仪方向数据，转换为方向文本与角度值。
4. 界面反馈：首页顶部展示方向文本，中部展示角度与圆盘指针联动效果。
5. 异常处理：传感器不可用时提示“当前设备不支持指南针功能”；方向数据不稳定时显示精度较差状态。

### 状态管理
- 正常状态：方向与角度实时刷新
- 加载状态：首次初始化方向数据
- 错误状态：传感器异常 / 权限受限
- 空状态：不适用

### 数据依赖
- 读取数据：设备方向传感器、磁场状态、定位信息（供麦加方向计算复用）
- 写入数据：无持久主数据，仅读取与本地设置联动
- 数据更新：按系统方向刷新频率持续更新

---

## 5.2 模块 B：精度检测与校准
- **模块目标**：在设备方向精度不佳时提供可执行的校准引导
- **用户价值**：提升方向读数可信度，降低误判
- **优先级**：P0

### 功能点列表
- 精度状态判断
- 磁场干扰提示
- 校准引导页
- 校准结果反馈

### 详细交互逻辑
1. 用户操作：在首页看到“精度较差”提示后点击“去校准”。
2. 系统响应：进入校准页并展示图示动作说明。
3. 数据流转：持续监测校准前后方向稳定度与磁场状态。
4. 界面反馈：页面提示用户按 8 字形移动手机，完成后显示“校准成功”或“请继续校准”。
5. 异常处理：长时间无法改善时提示远离磁铁、电池、电脑等干扰源；校准中断时允许返回首页并保留当前状态。

### 状态管理
- 正常状态：精度良好
- 加载状态：校准检测中
- 错误状态：校准失败 / 干扰持续
- 空状态：用户尚未触发校准

### 数据依赖
- 读取数据：磁场强度、方向稳定度、校准状态
- 写入数据：最近一次校准结果、提示关闭状态（如有）
- 数据更新：校准过程中实时更新

---

## 5.3 模块 C：麦加方向
- **模块目标**：根据用户当前位置提供相对麦加方位指引
- **用户价值**：满足明确的宗教朝向辅助场景
- **优先级**：P0

### 功能点列表
- 麦加方向入口
- 相对麦加方位显示
- 当前位置与目标方向提示
- 返回首页继续使用

### 详细交互逻辑
1. 用户操作：在首页点击“麦加方向”。
2. 系统响应：读取当前位置坐标并计算相对麦加的方位角。
3. 数据流转：使用定位数据计算目标方向，将结果映射到圆盘与箭头指示。
4. 界面反馈：展示“当前朝向 / 麦加方向 / 还需旋转角度”等信息，并高亮目标方向。
5. 异常处理：定位未授权时提示授权定位；定位失败时提供重试入口并保留说明态。

### 状态管理
- 正常状态：已获取方位并展示目标方向
- 加载状态：定位与方向计算中
- 错误状态：定位失败 / 权限关闭
- 空状态：首次进入待定位

### 数据依赖
- 读取数据：当前位置、设备方向、目标方位计算结果
- 写入数据：无核心持久数据
- 数据更新：定位成功后更新；方向变化时同步刷新指示

---

## 5.4 模块 D：设置与偏好
- **模块目标**：统一管理主题、语言、精度提示、反馈开关与协议入口
- **用户价值**：让用户根据使用习惯定制工具体验
- **优先级**：P0

### 功能点列表
- 主题切换
- 语言设置
- 精度提示开关
- 帮助与协议入口

### 详细交互逻辑
1. 用户操作：点击首页右上角“设置”。
2. 系统响应：进入设置页并加载当前偏好配置。
3. 数据流转：读取本地偏好；用户修改后写入本地设置。
4. 界面反馈：开关、列表项与当前选中状态即时更新；返回首页后视觉立即生效。
5. 异常处理：写入失败时提示“设置保存失败，请重试”；某些系统能力关闭时展示说明文案。

### 状态管理
- 正常状态：配置可编辑
- 加载状态：设置读取中
- 错误状态：本地写入失败
- 空状态：首次安装显示默认配置

### 数据依赖
- 读取数据：本地偏好、订阅状态、语言配置
- 写入数据：主题、语言、提示开关、帮助入口状态
- 数据更新：用户修改后即时更新

---

## 5.5 模块 E：订阅与会员
- **模块目标**：通过月订 / 年订提供高级权益解锁
- **用户价值**：让重度用户获得更完整、更纯净的体验
- **优先级**：P0

### 功能点列表
- 月订 / 年订购买
- 恢复购买
- 权益对比展示
- 高级主题与去广告解锁

### 详细交互逻辑
1. 用户操作：点击首页“升级 Pro”或在设置 / 高级主题入口进入订阅页。
2. 系统响应：拉取 SKU 并展示价格、权益和推荐方案。
3. 数据流转：读取 IAP SKU 配置，购买成功后刷新本地订阅状态与权益开关。
4. 界面反馈：展示已选套餐、高亮推荐项、购买成功提示与权益解锁结果。
5. 异常处理：SKU 加载失败时显示重试；支付取消时提示未完成购买；恢复失败时提示稍后再试。

### 状态管理
- 正常状态：SKU 正常展示
- 加载状态：商品拉取 / 购买处理中
- 错误状态：支付失败 / 拉取失败
- 空状态：无商品配置

### 数据依赖
- 读取数据：IAP SKU、订阅状态、本地权益开关
- 写入数据：订阅状态缓存、权益解锁状态
- 数据更新：购买 / 恢复购买后更新

- SKU 数量：2
- 支付方式：Apple IAP

## 9.2 SKU 定义
- SKU 1：Digital Compass Pro Monthly
  - 价格：US$2.99 / 月
  - 权益：去广告、高级主题、扩展圆盘样式、后续高级功能优先开放

  - 价格：US$19.99 / 年
  - 权益：包含月订全部权益，按年计费更优惠

## 9.3 免费版与 Pro 版差异
- 免费版：基础指南针、基础校准、基础麦加方向、基础设置
- Pro 版：去广告、高级主题、高级方向样式、更丰富视觉方案、后续高级功能扩展

## 9.4 订阅入口
- 首页“升级 Pro”按钮

- 高级主题 / 高级样式受限操作触发页

## 9.5 订阅触发规则
- 用户点击受限高级能力时直接进入订阅页
- 用户完成基础体验后可轻提示展示升级入口
- 已订阅用户不再弹出升级引导

## 9.6 恢复购买
- 设置页与订阅页提供“恢复购买”入口
- 恢复成功后立即刷新权益状态
- 恢复失败时给出明确提示与重试路径

# 10. 应用内协议与法务文档模块

## 10.1 模块目标
- 为 App 内提供完整的隐私协议、用户服务条款与订阅条款入口
- 方便开发直接接入，满足上架审核与用户可访问性要求

## 10.2 页面结构
- 隐私协议页
- 用户服务条款页
- 订阅条款 / 自动续费说明页

## 10.3 接入要求
- 设置页提供统一入口
- 订阅购买页提供订阅条款入口
- 文档页支持滚动阅读与返回
- 正式上线前需替换为法务确认版本

## 10.4 隐私协议（草案要点）
- 说明应用会读取设备方向传感器与必要的定位信息
- 定位仅用于麦加方向等依赖方位计算的功能
- 不建立账号系统，不做云同步
- 不主动收集与功能无关的个人信息

## 10.5 用户服务条款（草案要点）
- 产品为工具辅助类应用，方向结果受设备状态和环境影响
- 用户需在安全、合法场景下使用本应用
- 开发方有权更新功能、价格与条款内容

## 10.6 订阅条款（草案要点）
- 订阅将通过 Apple 账户扣费并自动续费
- 用户可在系统订阅管理中取消自动续费
- 取消后权益持续至当前计费周期结束
- 恢复购买适用于同 Apple ID 下的有效订阅记录
